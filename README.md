# 💛🐦‍⬛  Game Hub 

Welcome to the ultimate custom web-game portal! This project features a collection of beautifully designed, interactive puzzle and board games tailored specifically for Pillu (⚡) and Crow (🐦‍⬛). 

This lightweight, mobile-friendly project runs entirely in a single HTML file—no heavy frameworks or backend required.

## 🎮 The Games

1. **Meow-sweeper 😻:** A cute twist on the classic Minesweeper logic. Find the safe cuddles, and use the 💖 flag to mark the spicy hisses!
2. **Tic-Tac-Pika ❌⭕:** Classic Tic-Tac-Toe featuring custom Pillu and Crow avatar pieces. Play against a friend (hot-seat) or challenge the Bot.
3. **Connect Crows 🧶:** A strategy drop-down game. Connect four of your custom pieces in a row to win! (Pass-and-play or Bot mode).
4. **O-pika-llo ☯️:** A strategic board game (Reversi/Othello). Sandwich your opponent's pieces to flip them to your emoji. (Pass-and-play or Bot mode).
5. **Coin Toss 🪙:** Can't make a decision? Call it in the air!
6. **Guess Num 🔢:** A quick logic game. Try to guess the randomly generated number between 1 and 100 using higher/lower hints.
7. **Pika-dle 🔤:** A custom Wordle clone! Guess the secret 5-letter word in 6 attempts.
8. **Su-meow-ku 🧩:** Classic Sudoku puzzle. Click the blank squares repeatedly to cycle through the numbers 1-9 to solve the grid.
9. **Waff-meow 🧇:** A fun letter-swapping game! Click two letters to swap them around until they form intersecting valid words on the grid.

## ✨ Features

* **Fully Responsive:** Includes a mobile-friendly hamburger menu for easy navigation on small screens.
* **Custom Imagery:** Uses personalized `pikachu.png` and `crow.png` character sprites.
* **Dark Mode 🌙:** Seamless toggling between a soft "Baby Pink" theme and a "Neon Fuchsia Cyber-Cat" theme.
* **Victory Effects:** Gorgeous CSS/JS particle animations! Expect confetti and floating bubbles when you win, and an explosion of angry emojis if you lose.

## 🚀 Deployment (GitHub Pages)

Because this project is built natively with HTML, CSS, and JS, it can be deployed on GitHub Pages in seconds for free:

1. Create a repository on GitHub (e.g., `pillumeowster`).
2. Upload the `index.html` file into the main branch.
3. Upload your custom `pikachu.png` and `crow.png` image files into the same folder.
4. Go to **Settings > Pages**.
5. Set the Source to the `main` branch.
6. Wait 1-2 minutes, refresh, and visit your live link!

Enjoy playing! ⚡💖
